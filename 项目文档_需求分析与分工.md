## 一、项目概述

### 1.1 项目背景

本项目是企业项目实训课程的选题之一，旨在通过ROS2框架和Gazebo仿真环境，实现移动机器人搭载的多自由度机械臂完成自动化抓取与递送任务。该项目涵盖机器人运动控制、计算机视觉、任务调度等多个技术领域，具有较强的综合性和实践性。

### 1.2 项目目标

控制移动机器人搭载的多自由度机械臂，结合视觉定位，完成以下整套流程：
1. 目标物体识别与定位
2. 机械臂运动规划与控制
3. 物体抓取操作
4. 定点递送与放置
5. 机械臂复位

### 1.3 技术栈

| 技术领域 | 具体技术 |
|---------|---------|
| 开发框架 | ROS2 (Galactic+) |
| 仿真环境 | Gazebo Classic + wpr_simulation2 |
| 编程语言 | Python 3.8+ |
| 视觉处理 | OpenCV 4.5+ |
| 机械臂模型 | WPR1五轴机械臂 |
| 运动规划 | 自定义IK求解器 |

### 3.2 模块设计

#### 3.2.1 视觉识别模块 (vision_node)

**功能**：目标检测与位姿估计

**核心算法**：
1. 图像预处理（去噪、色彩空间转换）
2. 阈值分割（HSV色彩空间）
3. 轮廓检测与筛选
4. 位姿估计（像素坐标 → 机器人坐标系）

**输入**：摄像头图像流

**输出**：目标物体位姿（PoseStamped）

#### 3.2.2 机械臂控制模块 (arm_controller)

**功能**：逆运动学求解与关节控制

**核心组件**：
- IKSolver：逆运动学求解器，支持多种PTP模式
- ArmController：ROS2节点，处理运动控制逻辑

**运动模式**：
- MOVJ_XYZ：关节运动（最快）
- MOVL_XYZ：直线运动（保持末端直线）
- JUMP_XYZ：抬升运动（先抬升再移动）

**输入**：目标位姿、任务指令

**输出**：关节状态、目标到达状态

#### 3.2.3 夹爪控制模块 (gripper_controller)

**功能**：夹爪开合控制

**控制逻辑**：
- 接收GRASP命令 → 闭合夹爪
- 接收DELIVER命令 → 打开夹爪
- 接收RESET命令 → 打开夹爪

**输入**：任务指令

**输出**：夹爪状态

#### 3.2.4 任务调度模块 (task_scheduler)

**功能**：协调各模块完成完整流程

**状态机**：
```
IDLE → DETECTING → MOVING_TO_OBJECT → GRASPING → 
      MOVING_TO_TARGET → RELEASING → RETURNING → IDLE
```

**输入**：视觉目标、关节状态、夹爪状态

**输出**：任务指令、任务状态

#### 3.2.5 云台控制模块 (pan_tilt_controller)

**功能**：扩展视觉系统视野

**工作模式**：
- 自动扫描模式：云台自动扫描环境
- 目标追踪模式：根据视觉检测调整角度

**输入**：视觉目标位姿

**输出**：云台状态

## 五、技术难点与解决方案

### 5.1 逆运动学求解

**难点**：5轴机械臂的IK求解复杂度高

**解决方案**：
- 采用几何方法简化求解
- 实现多种PTP模式适配不同场景
- 使用梯形速度曲线实现平滑运动

### 5.2 视觉定位精度

**难点**：单目视觉的深度估计不准确

**解决方案**：
- 使用已知物体尺寸进行深度估计
- 结合机械臂反馈进行位置校正
- 实现目标到达检测机制

### 5.3 模块间同步

**难点**：多个ROS2节点的协调控制

**解决方案**：
- 设计清晰的状态机
- 使用话题消息进行异步通信
- 实现超时保护机制
